Otimize SEO para AltheiaSite. Argumento: texto ou página alvo + keyword principal.

**Input:** $ARGUMENTS

**Passos:**
1. Identifique keyword principal e secundárias (LSI) relacionadas ao argumento
2. Gere as meta tags otimizadas (title ≤60 chars, description ≤160 chars) no estilo Althéia (luxo, skincare)
3. Sugira heading hierarchy (H1 → H2 → H3) com keywords naturalmente integradas
4. Verifique densidade de keyword (alvo: 1-2%)
5. Gere schema markup JSON-LD adequado (Product, BreadcrumbList, Organization)
6. Liste internal links sugeridos para outras páginas do site

**Output estruturado:**
- `metadata` object TypeScript pronto para colar no arquivo Next.js
- Schema JSON-LD completo
- Sugestões de alt text para imagens
- Checklist on-page SEO

**Paleta Althéia:** tom sofisticado, palavras como "luminosidade", "ritual", "pureza", "ouro 24k", "élixir"
**Stack:** Next.js 16 App Router — usar `export const metadata: Metadata = {...}` no page.tsx
