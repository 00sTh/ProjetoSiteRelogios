Planeje estratégia de conteúdo SEO trimestral para Althéia. Argumento: período (ex: "Q2 2026") + foco (ex: "anti-idade", "hidratação", "lançamento produto").

**Input:** $ARGUMENTS

**Gere plano completo:**

1. **Keyword research**
   - 10 keywords principais (volume alto, concorrência média)
   - 20 keywords de cauda longa (nicho, intenção de compra)
   - Keywords sazonais do período

2. **Calendário editorial (12 semanas)**
   - Blog posts (2x/mês): título + keyword alvo + tipo (educativo/comparativo/aspiracional)
   - Páginas de categoria/coleção a criar
   - Produtos a otimizar

3. **Estratégia de link building**
   - Parceiros potenciais (beauty bloggers, revistas)
   - Guest posts sugeridos

4. **Métricas de sucesso**
   - KPIs: impressões, cliques, posição média, conversão
   - Ferramentas: Google Search Console, Analytics

5. **Quick wins (primeiras 2 semanas)**
   - Páginas existentes para otimizar agora
   - Meta tags a corrigir
   - Schema markup faltando

**Output:** documento estruturado + tabela calendário + lista de ações priorizadas.
